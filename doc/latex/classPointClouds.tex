\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classPointClouds}{Point\+Clouds}} class manages the contents of two Point\+Cloud objects, used to keep track of detected points and camera poses. This class can be interfaced to add and remove points from the clouds, as well as to request a screen refresh from the Easy3D viewer. 

Example usage\+:

\mbox{\hyperlink{classPointClouds}{Point\+Clouds}} clouds(points, poses); clouds.\+Add\+Point(0.\+0, 1.\+0, -\/0.\+5) // Adds to the {\ttfamily points} point cloud clouds.\+Add\+Point(2.\+0, -\/1.\+0, 0.\+0, true) // Adds to the {\ttfamily poses} point cloud clouds.\+Update\+View(); clouds.\+Add\+Poinst\+Mat\+Update($<$vector of cv\+::\+Mat objects$>$); clouds.\+Clear\+All();
