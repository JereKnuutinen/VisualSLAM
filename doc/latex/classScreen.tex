The screen class is used to interface with the viewer output. It manages the state of the UI, holds handles to the Open\+GL buffers used by Easy3D, and other things required to keep the visualization running. The screen class does NOT keep track of the point clouds used by the program. That responsibility is in the \mbox{\hyperlink{classPointClouds}{Point\+Clouds}} class.
